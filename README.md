# Mini Quotes
名人名言微信小程序


## 开发环境
- 准备 sandbox (要求 pub key 已经在 GitHub 账户 Profile 设置好)

```bash
cd ~/
mkdir sandbox
cd sandbox
git clone git@github.com:sim4d/mini-quotes.git mini-quotes
```

- 下载 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 从上述目录导入项目
